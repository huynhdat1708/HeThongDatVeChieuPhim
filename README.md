# Hệ Thống Đặt Vé Chiếu Phim

## Giới thiệu

Ứng dụng quản lý đặt vé rạp chiếu phim bằng C++.

Chức năng:

1. Hiển thị sơ đồ ghế
2. Đặt vé
3. Hủy vé
4. Tìm kiếm khách hàng
5. Xem vé đã đặt
6. Thống kê doanh thu


## Cấu trúc dữ liệu sử dụng

### 1. BST

Lưu khách hàng.

Chức năng:
- Thêm khách hàng
- Tìm kiếm theo ID


### 2. Priority Queue

Quản lý danh sách chờ.

Độ ưu tiên:

1. VIP
2. Người cao tuổi
3. Khách thường


### 3. Stack

Lưu lịch sử giao dịch:

- BOOK
- CANCEL


### 4. 2D Array

Quản lý sơ đồ ghế.
